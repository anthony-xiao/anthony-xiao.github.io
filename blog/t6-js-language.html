Javascript Language
Parentheses ()

  • When you are creating a function function new () {} or when you are excecuting the function. The Parentheses can either be left empty or have an argument(s) depending on the situation.
  • When there is an if statement if (condition)
  • When there is a for loop for (condition)
  • When there is a math equation i = (10+1)*20

Brackets []

  • When you are creating an array new array = ["item1", "item2"
  • To access a property for an object new.[firstProperty]
  • Place to enter in the index number when looking into an array list[2]

Braces {}

  • After the brackets for creating a function, within the braces you write the codes for the what the function will do. function new () {do this}
  • When creating an object you can use the braces to include the properties. var object = {style: new, place: Auckland}

Single quote '' and double quote ""

In Javascript both of these can be used interchangeably. There one case you would think about using one over another is if you have a ' or " within your string.

  • When you are writing a string
  • When you are accessing the property of an object
  • When you are printing something on console.log
