Scope
What 'scope' is, and how it works in JavaScript.

In basic terms scope means how visible your variables are. There are 2 different scopes in JavaScript. Global and local. Global is when your variables are visible from anywhere in your code, hence the word global. Also in when variables are created the order in which they sit in the code can be very important. If a variable is created without initialising it, and it is called for, it will return undefined instead of the actual element. Also functions are always parsed first prior to running the rest of the code. Local scope is when variables are created in a function and only accessible within the funciton it was created in. If within the function there is another function all the variables witin the function's parenthesis will be accessble to any functions within the same parenthesis.
